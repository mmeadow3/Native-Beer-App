# Native-Beer-App

##### This is a Beer Finder App made using HTML, jQuery, and Electron
##### Styled with basic CSS and Photon
##### Uses npm modules: request and request-promise
##### Uses [breweryDB](http://www.brewerydb.com/) as the API requested
